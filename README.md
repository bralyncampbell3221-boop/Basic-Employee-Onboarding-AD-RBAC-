# Basic Employee Onboarding (AD)(RBAC)

## Problem Statement
The problem in this project was related to a fictional healthcare company called Northstar Medical Group. As the organization grew, its identity infrastructure became disorganized due to poor Active Directory management by a third-party MSP. Users were assigned permissions manually, Organizational Units lacked proper structure, and Role-Based Access Control (RBAC) was not consistently implemented, creating security risks and potential HIPAA compliance concerns.


## Solution Overview
The solution was to build and organize a secure Active Directory environment from the ground up. I created the NMG.com domain, designed Organizational Units, implemented Role-Based Access Control (RBAC) using security groups, configured Group Policy, and simulated a real-world support ticket by investigating and resolving an Active Directory access issue while documenting the entire process.
